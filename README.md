## Installing

### Core

```shell
[Package Manager]
PM> Install-Package Exomia.Vulkan.Api.Core
```

### Windows

```shell
[Package Manager]
PM> Install-Package Exomia.Vulkan.Api.Windows
```

### Linux (comming soon)

```shell
[Package Manager]
PM> Install-Package Exomia.Vulkan.Api.Linux
```

### Beta

```shell
[Package Manager]
PM> Install-Package Exomia.Vulkan.Api.Beta
```
